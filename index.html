Using the Template

Editing …

  1. Find the file oddExercise_template2.xml in the root level of your ODD_exercise/ directory.
  2. Make a copy of it (named whatever you want) in the same directory!
  3. Open the copy in oXygen
  4. OPTIONAL: put the right metadata in the <teiHeader> (after all, your document was not written by some named “[you!]” :-)
  5. OPTIONAL: Write (multiple chapters of) prose documentation in the <body>
  6. Edit the fields surrounded by square brackets, or delete entire specification elements in the <schemaSpec>

Check validity early and often; save early and often.
